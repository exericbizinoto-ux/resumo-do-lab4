# Resumo do Lab: Dominando Conceitos de Cloud e Infraestrutura

Este repositório foi criado como parte de um desafio da DIO para documentar os aprendizados adquiridos durante o laboratório prático.

## 🚀 O que eu aprendi

Durante o desenvolvimento deste lab, explorei conceitos fundamentais que são pilares para qualquer profissional de TI moderno:

* **Fundamentos de Cloud Computing:** Compreensão dos modelos de serviço (IaaS, PaaS, SaaS) e a importância da escalabilidade e disponibilidade.
* **Gestão de Identidade e Acesso:** A importância de políticas de segurança e o gerenciamento de usuários em ambientes de nuvem (como o Azure AD/Entra ID).
* **Infraestrutura como Serviço:** Noções de como provisionar e gerenciar recursos computacionais de forma eficiente.
* **Segurança e Governança:** Práticas essenciais para proteger dados e garantir que a infraestrutura esteja em conformidade com as normas do setor.

## 🛠️ Ferramentas Utilizadas

* **GitHub:** Para versionamento de código e documentação.
* **Azure (ou plataforma específica do lab):** Exploração do portal e serviços de nuvem.
* **Markdown:** Para a estruturação deste resumo.

## 🧠 Conclusão

O lab foi fundamental para conectar a teoria à prática, permitindo uma visão clara de como as grandes empresas gerenciam seus parques tecnológicos atualmente. O próximo passo é aplicar esses conceitos em projetos mais complexos e buscar certificações na área.
